# 🇰🇷 대한민국 경제 지표 분석 (2000-2023)

이 프로젝트는 2000년부터 2023년까지 대한민국의 **경제성장률**과 **청년실업률** 데이터를 분석하여 시각화하는 파이썬 프로젝트입니다.

##  프로젝트 구조
- `data/`: 통계청 등에서 다운로드한 엑셀 원본 데이터가 위치합니다.
- `src/`: 데이터를 불러와 그래프를 그리는 파이썬 소스 코드가 있습니다.
- `requirements.txt`: 프로젝트 실행에 필요한 라이브러리 목록입니다.

##  주요 기능
1. **경제성장률 분석**: 연도별 GDP 성장률 추이를 막대그래프로 시각화
2. **청년실업률 분석**: 연도별 청년 실업률 변동 추이 시각화
3. **자동 폰트 설정**: Windows, Mac, Linux 환경에 맞춰 한글 폰트 자동 적용

##  실행 방법

1. 필수 라이브러리 설치
   ```bash
   pip install -r requirements.txt

###  적용 가이드
1. 위 **폴더 구조**대로 폴더와 파일을 생성합니다.
2. 각 **코드 블록**의 내용을 해당 파일에 붙여넣습니다.
3. 가지고 계신 **엑셀 파일 2개**를 `data` 폴더에 넣습니다.
4. 터미널에서 `git init`, `git add .`, `git commit -m "Initial commit"` 명령어로 깃을 시작하고, 깃허브에 푸시(Push)하시면 됩니다.
